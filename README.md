<h1>🌍 WanderLust – Airbnb Clone</h1>

<p><strong>A Full Stack Web Application for Listing and Exploring Stays</strong></p>

<p>
WanderLust is a full-stack web application inspired by Airbnb, where users can 
create, explore, and review property listings. It provides a complete experience 
including authentication, listing management, and user-generated reviews.
</p>

<h2>🚀 Getting Started (Local Setup)</h2>

<h3>1. Clone the Repository</h3>
<pre>
git clone https://github.com/Vishal-Kumar2006/Wanderlust.git
cd Wanderlust
</pre>

<h3>2. Install Dependencies</h3>
<pre>
npm install
</pre>

<h3>3. Setup Environment Variables</h3>
<p>Create a <code>.env</code> file in the root directory and add:</p>

<pre>
MONGO_URI=your_mongodb_connection_string
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
SESSION_SECRET=your_secret_key
MAPBOX_TOKEN=your_mapbox_token
</pre>

<h3>4. Run the Application</h3>
<pre>
npm start
</pre>

<p>Open your browser at: <code>http://localhost:3000</code></p>
